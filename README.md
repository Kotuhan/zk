# Калькулятор прибутковості

React-калькулятор на основі Excel-файлу `Untitled spreadsheet.xlsx`.

## Логіка

- Витрати йдуть зверху вниз і зменшують залишок послідовно.
- **Маркетинг** рахується **від суми, яка залишилась після витрат на менеджерів**.
- Усі інші витрати рахуються від **загальної вартості** (дохід = ціна * кількість).
- Кожну витрату можна задавати у **%** (повзунок) або у **грн** (поле суми). Поля звʼязані.

## Запуск

```bash
npm install
npm run dev
```

## Збірка

```bash
npm run build
npm run preview
```
